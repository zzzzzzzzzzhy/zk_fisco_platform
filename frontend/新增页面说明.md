## 📝 使用说明

### 访问页面

1. **排行榜列表**
   ```
   http://localhost:8081/leaderboards
   ```
   - 选择竞赛查看排行榜
   - 切换公榜/私榜
   - 创建榜单快照

2. **排行榜详情**
   ```
   http://localhost:8081/leaderboards/{id}
   ```
   - 查看排名详情
   - 提交异议

3. **异议管理**
   ```
   http://localhost:8081/leaderboard-appeals
   ```
   - 查看和审核异议

4. **奖金管理**
   ```
   http://localhost:8081/prize-management
   ```
   - 管理奖金池、分配、风控、发放

### 权限说明

- **普通用户**：
  - 可以查看排行榜
  - 可以提交异议

- **管理员**：
  - 所有普通用户权限
  - 创建和管理排行榜
  - 审核异议
  - 管理奖金池和分配
  - 执行风控和发放
